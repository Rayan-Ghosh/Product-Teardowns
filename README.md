# Product-Teardowns

A running collection of deep-dive product teardowns — each one applying structured frameworks (behavioural economics, competitive strategy, compliance/SWOT analysis) to break down what a real product gets right, where it's leaking users, and what a lean team could ship to fix it.

Every teardown follows the same discipline: **observed behavior → root cause → shippable fix**, not surface-level opinion.

## 🔍 Subspace.money — At a Glance

A product intern assignment analyzing how a profitable, zero-VC fintech app (₹36.5Cr ARR FY25) is still losing high-intent users — despite genuine product-market fit.

**Key numbers:**

| Metric | Value |
|---|---|
| ARR (FY25, Zero VC) | ₹36.5Cr |
| Play Store Rating | 3.6★ |
| 1-Star Reviews | ~38% |
| India Paid OTT Subscriber Base | 96M |

**Four pillars analyzed:**

1. **Features & UX** — registration walls, hidden anchor pricing, and unchecked admin impunity over the escrow wallet
2. **Competitor Analysis** — misdirected positioning against a collapsing rival while a real technological moat sits unused
3. **Brand & Compliance** — AI-hallucinated blog content creating live regulatory exposure, plus a stale public identity
4. **Collaborations** — five zero-CAC partnership vectors (dating apps, quick-commerce, transit, fintech, connected TV) for distributing the underused escrow/negotiation infrastructure

Every recommendation is scoped to what a 3-person team could ship in under 60 days.
